# node-rent-platform

## 开发环境

> node -v >=6.9.0

## 功能
1. 完备的后端模板渲染
2. 路由模块
3. 404、500错误处理
4. service、controller分层
5. 适配不同的开发环境 => dev,test,prod
6. 不同环境的动态错误日志生成

## 业务分层

> API 收集处理客户端请求数据，将处理好的数据发给service
> SERVICE 主业务层，根据业务逻辑分块，去不同的soa请求数据进行封装反馈给API。
> SOA 每个SOA单独模块封装，以HTTP协议请求数据，不做数据封装
